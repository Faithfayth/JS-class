Arrow Functions

These are a more concise way to write functions in JavaScript.

And they do not have their own 'this' context.

JSON file storage

JSON stands for JavaScript Object Notation. It is a lightweight data-interchange format that is easy for humans to read and write, and easy for machines to parse and generate. JSON is often used to store and exchange data between a server and a web application.

Local Storage

Local Storage is a web storage API that allows you to store data in the browser. It is a key-value store that can be accessed using JavaScript. Local Storage is persistent, meaning that the data will remain even after the browser is closed.

Local storage works with strings and not with objects.
